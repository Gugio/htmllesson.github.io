Глобальні атрибути:

Атрибут Властивість
accesskey="s"
(цифра (0-9) або латинська буква (a-z))		 Перехід до елемента з допомогою комбінації клавіш.
Internet Explorer: Alt + S / Chrome: Alt + S / Opera: Shift + Esc, S / Safari: Alt + S / Firefox: Shift + Alt + S (використовується тільки для <a>, <area>, <button>, <input>, <label>, <legend>, <textarea>)
class = "my-super-class another-class" Атрибут класу вказує один або кілька імен класів для елемента.
contenteditable = "false / true" <span> Вказує, чи змінюється вміст елемента чи ні. <span>
data-* Використовується для зберігання спеціальних даних, приватних для сторінки або програми.
dir = "ltr / rtl" <span dir="ltr">Вказує напрямок тексту для вмісту елемента.</span>
<span dir="rtl">Вказує напрямок тексту для вмісту елемента.</span>
<span dir="rtl" style="unicode-bidi: bidi-override;">Вказує напрямок тексту для вмісту елемента.</span>
draggable="true / false" Вказує, чи елемент можна перетягнути чи ні.
dropzone="copy / move / link" Визначає, що робити з перетягувати користувачем даними - скопіювати їх, перетягнути або зв'язати - в момент, коли дані кидаються на елемент.
copy - після зняття даних буде отримана копія перетягнутих даних.
move - перетягнутий обєкт переміститься в нове місце.
link - після переміщеня обєкта буде отримана ссилка на нього.
hidden Приховує елемент (властивість display: none;).
id = "my-id" Задає ідентифікатор елементу.
lang = "en" Визначає мову вмісту елемента для кодування.
spellcheck = "false / true" Вказує браузеру, чи перевіряти правопис і громатику чи ні.
style = "color: orang; font-size:18px;" Використовується для задання стилю елеманту з допомогою правил CSS.
tabindex = "ціле позитивне число" Задає порядок отримання фокуса при переході між елементами з допомогою Tab
title = "додаткова інфа" Вказує додаткову інформацію про елемент, у вигляді спливаючої підказки.
