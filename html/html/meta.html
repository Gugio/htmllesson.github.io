<meta name="viewport" content="width=device-width, initial-scale=1.0">
<meta name="keywords" content="HTML, CSS">

<meta name="author" content="Jumshut Ravshnovich">
</head>

< meta name="viewport" content="width=device-width, initial-scale=1.0, maximum-scale=1.0, user-scalable=no" >

width

Ціле число (від 200px — 10,000px) або “device-width”. Задає ширину viewport. Якщо ширина viewport не задана, то: для мобільного: Safari - 980px Opera — 850px Android WebKit — 800px IE — 974px

height

Ціле число (від 223px до 10,000px) або “device-height” Задає висоту viewport. В 99% випадках він не потрібний.

initial-scale

Доступні значення (від 0.1 до 10). 1.0 — не маштабувати. Задає маштаб сторінки. Збільшуючи значення - збільшується маштаб.

user-scalable

Доступні значення no або yes Задає можливість змінювати маштаб сторінки. По замовчуванню - yes.

minimum-scale і maximum-scale

Доступні значення (від 0.1 до 10). 1.0 — не маштабувати. Визначає мінімальний і максимальний маштаб viewport. За замовчуванням в мобільному наприклад Safari minimum-scale = "0.25", maximum-scale = “1.6”.

Визначення ключових слів для пошукових систем:

<meta name="keywords" content="HTML, CSS">

Визначити опис вашої веб-сторінці:

<meta name="description" content="Free Lessons">

Визначення автора сторінки:

<meta name="author" content="Jumshut Ravshnovich">

Оновити документ кожні 30 секунд:

<meta .http-equiv.="refresh" content="30">
