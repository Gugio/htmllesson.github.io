Input

<input type="тип інпута" title="опис">

Тег <label> встановлює зв'язок між певною міткою, в якості якої зазвичай виступає текст, і елементом форми (<input>, <select>, <textarea>). Такий зв'язок необхідний, щоб змінювати значення елементів форми при натисканні курсором миші на текст. Крім того, за допомогою <label> можна встановлювати гарячі клавіші на клавіатурі і переходити на активний елемент подібно посиланнях.

Існує два способи зв'язування об'єкта і мітки. Перший полягає у використанні ідентифікатора id всередині елемента форми і надання його імені в якості атрибута for тега <label>. При другому способі елемент форми поміщається всередину контейнера <label>.

Атрибути:
accesskey="цифра (0-9) або латинська буква (a-z)" Перехід до елемента з допомогою комбінації клавіш.
Internet Explorer: Alt + S / Chrome: Alt + S / Opera: Shift + Esc, S / Safari: Alt + S / Firefox: Shift + Alt + S
(використовується тільки для <a>, <area>, <button>, <input>, <label>, <legend>, <textarea>)
autocomplete = "on / of" Включає або відключає автозаповнення. type="text / password / email / search / url / tel"
autofocus Установлює фокус в поле форми.
accept = " image/*,audio/*,video/* " Фільтр на типи файлів, які ви можете відправляти. type="file"
alt = "опис" Альтернативний текст для кнопки з картинкою. Обов'язковий. type = "image"
border = "5" Товщина рамки навколо картинки. type = "image"
checked Активує type = "radio / checkbox" type="checkbox / radio"
disabled Блокує доступо для зміни елемента.
form = "ID форми" Зв'язує поле з формой по її ID.
formaction = "handler.php" Визначає адресу обробника форми. Аналог атрибута форми action type="submit"
list = "ID списка" Вказує на список варіантів, які можна вибирати при введенні тексту.   <input list="ID">
<datalist id="ID">
 <option value="Текст1">
 <option value="Текст2">
</datalist>
max = "число / дата(2017-09-27)" Максимальне значення для введення числа або дати. type = "number / range / date"
maxlength = "число" Максимальна кількість символів дозволених в тексті. type = "text / password"
min = "число / дата(2017-09-27)" Мінімальне значення для введення числа або дати. type = "number / range / date"
multiple Дозволяє завантажити кілька файлів одночасно, або декілька поштових адресів (які повинні бути розділені комою) type="file / email"
name = "iм'я" Ім'я поля, призначене для того, щоб обробник форми міг його ідентифікувати. Для type = "radio" додаткова функція вибору тільки одного значення.
placeholder = "текст-приклад" Виводить текст-підказку в полі вводу. type = "text / email / url / datetime / tel / search / number"
readonly Встановлює, що поле не може змінюватися користувачем. type = "text / email / url / datetime / tel / search / number"
required Обов'язкове для заповнення поле.
size = "ціле позитивне число" Ширина текстового поля в символах. type = "text / email / url / datetime / tel / search / number"
src = "url" Адреса графічного файлу для поля із зображенням. type = "image"
step = "ціле або дробне число" Крок збільшення для числових полів. type = "number / range"
tabindex = "ціле позитивне число" Визначає порядок переходу між елементами за допомогою клавіші Tab.
value = "значення" Залежно від типу елемента атрибут value виступає в наступній ролі:
  • для кнопок (input type = "button | reset | submit") встановлює текстовий напис на них;
  • для текстових полів (input type = "password | text") вказує попередньо введений рядок. Користувач може правити текст і вводити свої символи, але при використанні в формі кнопки Reset призначений для користувача текст очищається і відновлюється введений в атрибуті value;
  • для (input type = "checkbox | radio") унікально визначає кожен елемент, з тим, щоб клієнтська або серверна програма могла однозначно встановити, який пункт вибрав користувач.
  • для файлового поля (input type = "file") не робить вплив.

tabindex = "ціле позитивне число" Визначає порядок переходу між елементами за допомогою клавіші Tab.
wrap = "soft / hard" Параметры переноса строк:

soft - За замовчуванням. Довгий текст, який самостійно не поміщається в поле по ширині, буде автоматично перенесено на новий рядок, проте передаватися на сервер буде як один рядок. Натискання клавіші Enter встановлює перенесення тексту, який зберігається при відправці форми.

hard - Слова в полі переносяться механічно, щоб вони помістилися в розмір області, і при відправленні на сервер місця автоматичного переносу зберігаються. При цьому значенні обов'язково повинен бути присутнім атрибут cols.
