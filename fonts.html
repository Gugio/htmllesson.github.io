$base-font-data : (

@for $i from 1 through length($base-font-data) {
 @font-face {
  font-family: MyFontName;

 }
}

Колір

Колір використовується для зміни кольору тексту, фону, градіентів, тіней, границь та ін. Є кілька основних методів для задання кольору у CSS:
Назва кольору (red, green, blue ...)
Шістнадцяткова система (#ffffff, #000000, #db4e44 ...), (#fff, #000 ...)
'Функція' rgb(255, 255, 255) і rgba(255, 255, 255, 1)

  • Колір
    CSS визначає 145 назв кольорів. Від самих простих (red, green, blue, yellow...) і до спецефічних (lawngeen, orchid, crimson ...)
  • #66cc00 (шістнадцятирічний колір)
    Являє собою код кольору з шести символів, які починаються після знаку #

    Кожна пара символів представляє номер, від 0 до 255. Перші два - (red), наступні два - (green), остання пара - (blue)
    Для блоку нижче встановлено: .selector { background-color: #bb12ff; }

  • rgb()
    Система rgb() використовує 3 числа, які описують відносну к-сть червоно, зеленого, синього кольорів які у поєднанні можуть вивести будь-який колір. Числове значення кольору варіюється від 0 до 255.

    Наприклад для блоку нижче задано: .selector { background-color: rgb(205, 50, 255); }

  • rgba()
    Система rgba() ще одне значення - прозорість. Значення останнього варіюється від 0 до 1. Значення у 0.5 зробить колір напівпрозорим. Буква a - означає альфа-канал

    Для блоку нижче задано колір з прозорістю: .selector { background-color: rgba(205, 50, 255, 0.5); }

  • hsl()

    Наприклад для блоку нижче задано: .selector { background-color: hsl(285, 100%, 60%); }

  • hsla()

    Для блоку нижче задано колір з прозорістю: .selector { background-color: hsla(285, 100%, 60%, 0.5); }
