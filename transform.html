transform

CSS трансформації. Змінюють розмір, форму та положення елементів на сторінці.

Трансформуються елементи зі значенням display: block / inline-block; а також table-row, table-row-group, table-header-group, table-footer-group, table-cell, table-caption

Існує два види трансформації: 2D і 3D.

Функція трансформації transform

Властивітьзадає вид трансформації елементу. Властивість описується за допомогою функцій трансформації. Які зміщують його положення стосовно його теперішнього положення.

translateX(n) Зсуває елемент по осі Х. Приймає px/em/rem/%. % - береться від значення до якого використовується transition а не від батьківського елемента
translateX(20px)
translateY(n) Зсуває елемент по осі Y. Приймає px/em/rem/%.
translateY(-20px)
translate(x,y) Зсуває елемент по осям (x,y), відносно його положення. Додатнє значення - вправо по Х, або вниз по Y. Від'ємне значення - вліво по Х, або вверх по Y.
translate(20px, 20px)
scaleX(n) Масштабує елемент по ширині, роблячи його вужче/ширше. Якщо значення більше 1 то елемент стає ширшим, нижче 1 - вужчим
scaleX(0.8)
scaleY(n) Масштабує елемент по висоті, роблячи його вище/коротше. Якщо значення більше 1 то елемент стає вищим, нижче 1 - коротшим
scaleY(1.2)
scale(x,y) Масштабує елементи по осям. Значення від 1 до 0 - зменшують елемент. Перше значення масштабує по висоті, друг - по ширині.
scale(1.2, 0.5)
rotateX(градус) Використовується для обертання елементу довкола його X осі. Приймаються значення у градусах (360deg). Від'ємне значення обертає елемент проти годинникової стрілки, додатнє - за годинниковою стрілкою.
rotateX(180deg)
rotateY(градус) Використовується для обертання елементу довкола його Y осі. Приймаються значення у градусах (360deg). Від'ємне значення обертає елемент проти годинникової стрілки, додатнє - за годинниковою стрілкою.
rotateY(180deg)
rotate(градус) Використовується для обертання елементу довкола його центру. Приймаються значення у градусах (360deg). Від'ємне значення обертає елемент проти годинникової стрілки, додатнє - за годинниковою стрілкою.
rotate(180deg)
skewX(градус) Деформує сторони по осі Х
skewX(20deg)
skewY(градус) Деформує сторони по осі Y
skewY(-20deg)
skew(x-градус, y-градус) Деформація сторін елементу відносно осейю Якщо вказано одне значення, інше вираховується автоматично.
skew(20deg, -20deg)
matrix(a, b, c, d, x, y) Зміщує елементи та задає спосіб їх трансформації, дозволяє поєднати кілька функцій одночасно.У якості трансформації допускається поворот, масштабування, нахил, та зміна положення.
Значення А - змінює масштаб по горизонталі (scaleX). Задається від 0 до 1. Менше 1 - зменшує, більше 1 - збільшує
Значення B - деформує сторони по осі Y (skewY). Приймає числові значення. Від'ємне зсуває -вниз, додатнє - вверх.
Значення C - деформує сторони по осі X (skewX). Приймає числові значення. Від'ємне зсуває - вправо, додатнє - вліво.
Значення D - змінює масштаб по вертикалі (scaleY). Задається від 0 до 1. Менше 1 - зменшує, більше 1 - збільшує.
Значення Х - зміщує елемент по осі X (translateX). Приймає числові значення. Від'ємне зміщує - вліво, додатнє - вправо.
Значення Y - зміщує елемент по осі Y (translateY). Приймає числові значення. Від'ємне зміщує - вверх, додатнє - вниз.
matrix(1.2,0,0.5,1.2,50,0)
none Дефолтне значення. Відміняє трансформацію, якщо раніше вона була задана.

Точка трансформації transform-origin

Властивість дозволяє змістити центр початку. Дефолтне значення: transform-origin: center center або 50% 50%.

transform-origin: x y z; Пара значень. Задана за допомогою ключових слів, одиниць довжини px/em/rem/%. Визначає відносно якої частини елементу буде відбуватись трансформація. Використовується з rotate, scale і skew
transform-origin: top left

СSS переходи transition

СSS переходи дозволяють анімувати значення основні на інші, за заданий проміжок часу.

Зміна властивостей відбувється при якому-небудь івенті.

Переходи застосовуються до всіх елементів, включно з псевдоелементами :before :after.
transition: transition-property transition-duration transition-timing-function transition-delay

transition-property: none / all / властивості Визначає до яких властивостей буде застосована "таймінг" функція. all - дефолтне значення.
transition-duration: s / ms Задає час в секундах або мілісекундах, скільки повинна тривати анімація переходу до її завершення. За замовчуванням значення дорівнює 0s, це означає, що ніякої анімації немає, перехід відбувається миттєво.
Можна вказати кілька значень, перераховуючи їх через кому. Кожне значення застосовується до властивостей, заданих через transition-property.
transition-timing-function: ease / ease-in / ease-out / ease-in-out / linear /step-start / step-end / steps() / cubic-bezier() Являє собою математичну функцію, яка показує, як швидко за часом змінюється вказане через transition-property значення властивості. Початкова точка має координати 0.0, 0.0, кінцева - 1.0, 1.0, при цьому функція по осі ординат може перевищувати ці значення в більшу або меншу сторону.
ease - Анімація починається повільно, потім прискорюється і до кінця руху знову сповільнюється. Аналогічно cubic-bezier (0.25,0.1,0.25,1).
ease-in - Анімація повільно починається, до кінця прискорюється. Аналогічно cubic-bezier (0.42,0,1,1).
ease-out - Анімація починається швидко, до кінця сповільнюється. Аналогічно cubic-bezier (0,0,0.58,1).
ease-in-out - Анімація починається і закінчується повільно. Аналогічно cubic-bezier (0.42,0,0.58,1).
linear - Однакова швидкість від початку і до кінця.
step-start - Як такої анімації немає. Стильові властивості відразу ж приймають кінцеве значення.
step-end - Як такої анімації немає. Стильові властивості знаходяться в початковому значенні заданий час, потім відразу ж приймають кінцеве значення.
steps - Ступінчаста функція, що має задане число кроків. transition-timing-function: steps (<число>, start | end)
transition-delay: s / ms Bстановлює час очікування перед запуском ефекту переходу. Значення 0s запускає анімацію відразу ж. Негативне значення також включає анімацію без затримок, але може привести до зміни виду початку анімації.
Припустимо вказувати кілька значень, перераховуючи їх через кому. Кожне значення буде застосовуватися до властивості, заданому в параметрах transition-property.
