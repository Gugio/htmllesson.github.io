transform

CSS трансформації. Змінюють розмір, форму та положення елементів на сторінці.

Трансформуються елементи зі значенням display: block;, inline-block; а також table-row, table-row-group, table-header-group, table-footer-group, table-cell, table-caption

Існує два види трансформації: 2D і 3D.

Функція трансформації transform

Властивітьзадає вид трансформації елементу. Властивість описується за допомогою функцій трансформації. Які зміщують його положення стосовно його теперішнього положення.

none Дефолтне значення. Відміняє трансформацію, якщо раніше вона була задана.
matrix(a, c, b, d, x, y) Зміщує елементи та задає спосіб їх трансформації, дозволяє поєднати кілька функцій одночасно. У якості трансформації допускається поворот, масштабування, нахил, та зміна положення.
Значення А - змінює масштаб по горизонталі. Задається від 0 до 1. Менше 1 - зменшує, більше 1 - збільшує
Значення С - деформує сторони по осі Y. Приймає числові значення. Від'ємне зсуває -вниз, додатнє - вверх.
Значення B - деформує сторони по осі X. Приймає числові значення. Від'ємне зсуває - вправо, додатнє - вліво.
Значення D - змінює масштаб по вертикалі. Задається від 0 до 1. Менше 1 - зменшує, більше 1 - збільшує.
Значення Х - зміщує елемент по осі X. Приймає числові значення. Від'ємне зміщує - вліво, додатнє - вправо.
Значення Y - зміщує елемент по осі Y. Приймає числові значення. Від'ємне зміщує - вверх, додатнє - вниз.
translate(x,y) Зсуває елемент по осям (x,y), відносно його положення. Додатнє значення - вправо по Х, або вниз по Y. Від'ємне значення - вліво по Х, або вверх по Y.
translateX(n) Зсуває елемент по осі Х. Приймає числові/відсоткові значення.
translateY(n) Зсуває елемент по осі Y. Приймає числові/відсоткові значення.
scale(x,y) Масштабує елементи по осям. Значення від 1 до 0 - зменшують елемент. Перше значення масштабує по висоті, друг - по ширині.
scaleX(n) Масштабує елемент по ширині, роблячи його вужче/ширше. Якщо значення більше 1 то елемент стає ширшим, нижче 1 - вужчим
scaleY(n) Масштабує елемент по висоті, роблячи його вище/коротше. Якщо значення більше 1 то елемент стає вищим, нижче 1 - коротшим
rotate(градус) Використовується для обертання елементу довкола його центру. Приймаються значення у градусах (360deg). Від'ємне значення обертає елемент проти годинникової стрілки, додатнє - за годинниковою стрілкою.
skew(x-градус, y-градус) Деформація сторін елементу відносно осейю Якщо вказано одне значення, інше вираховується автоматично.
skewX(градус) Деформує сторони по осі Х
skewY(градус) Деформує сторони по осі Y
translateX(20px)
translateY(-20px)
translate(-20px, 20px)
scaleX(-1)
scaleY(1.2)
scale(1.2, 0.5)
rotate(720deg)
skewX(20deg)
skewY(-20deg)
skew(-20deg, 20deg)
matrix(1.2,0,0.5,1.2,50,0)

Точка трансформації transform-origin

Властивість дозволяє змістити центр початку. Дефолтне значення: transform-origin: center center або 50% 50%.

вісь Х (left, center, right, %) Пара значень. Задана за допомогою ключових слів, одиниць довжини px/em/rem... та відсоткових значень. Визначає відносно якої частини елементу буде відбуватись трансформація.

СSS переходи transition

СSS переходи дозволяють анімувати значення основні на інші, за заданий проміжок часу.

Зміна властивостей відбувється при якому-небудь івенті. Найчастіше спрацьовує псевдоклас :hover.

Переходи застосовуються до всых елементыв, включно з псевдоелементами :before :after.
