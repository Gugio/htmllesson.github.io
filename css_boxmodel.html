border-box;		 Властивість box-sizing використовується для зміни блочної моделі CSS застосовуваної за замовчуванням, за допомогою якої обчислюються ширина і висота елементів.
border-box - розмір задається враховуючи контент, padding i border-width
content-box - ширина задається враховуючи тільки контент.

padding-box;
content-box;		 вказує, як поширюється фон елемента, колір або зображення:
border-box - фон займає весть прості враховуючи ширину бордера і відображається під бордером.
padding-box - фон займає простір контенту і padding.
content-box - фон відображається тільки під контентом
