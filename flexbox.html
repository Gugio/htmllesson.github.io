flexbox

flexbox це порівняно нова технологія, яка підтримується всіма браузерами з середини 2016 року (IE 11 частково).

Flexbox надає інструментарій для швидкого створення складних. гнучких макетів, та функцій, які було важко реалізувати традиційними медотами CSS.

Наприклад:

  • Вертикальне вирівнювання дочірнього елементу
  • Розташування елементів, таким чином, щоб вони розділяли між собою всю доступну ширину/висоту, не залежно від ширини/висоти батьківського контейнеру
  • Створення колонок однакової висоти, без залежності від кількості контенту

Flexbox не являє собою блочний контейнер, тому для внутрішніх блоків не спрацюють властивості float, clear, vertical-align. Також не спрацююсь і псевдоелементи ::first-line, ::first-letter.

Щоб почати використовувати магію flexbox, достатньо визначити правило відображення елементу:

Після встановлення потрібного значення, діточки flexbox-у, вирівнюються по головній осі, колонками однакової висоти, рівній висоті батька. При цьому блочні, або строкоі елементи поводять себе однаково добре, і не хуліганять, їхня ширина дорівнює ширині контенту.

Вирівнювання елементів по горизонталі justify-content

Властивість вирівнює флекс-елементи по-ширині контейнеру, розділяючи вільний простір між елементами. Для їх вирівнювання по-вертикалі використовують align-content.

flex-start Значення дефолтне. Елементи позиціонуються від початку контейнеру
flex-end Елементи позиціонуються відносно правого краю контейнеру
center Елементи "піджимаються" до середини та центруються
space-between Перший блок розміщується на початку контейнеру, а останній у кінці, блоки між ними, розподіляють залишившийся простір рівномірно між собою.
space-around Всі блоки рівномірно розділяють між собою простір флекс-контейнеру
initial Встановлює значення за замовчуванням
inherit Наслідує від батьківського елементу

Синтаксис:

flex-direction напрямок головної осі

Властивість визначає, яким чином флекс-елементи вкладаються у флекс-контейнер. Флекс-елементи можуть розташовуватись у двох напрямках: горизонтально, як строки, або вертикально, як колонки.

row Значення дефолтне. Елементи розташовуються від краю флекс-контейнеру у одну строку
row-reverse Елементи розташовуються від протилежного краю флекс-контейнеру у одну строку, але у зворотньому напрямку. Ніби відзеркалення.
column Так як дане значення змінює головну вісь контейнера з горизонтальної на вертикальну, то і флекс-елементи вистраюються один під одним у колонку.
column-reverse Елементи вистраюються у колонку один під одним, але у зворотньому напрямку як і у row-reverse, але як колонка. Ну ви поняли ;-)
initial Встановлює значення за замовчуванням
inherit Наслідує від батьківського елементу

row row-reverse column column-reverse

Синтаксис:

flex-wrap багатостроковість флекс-елементів

Властивість визначає, яким чином флекс-елементи будуть вистраюватись, в одну строку, або переноситись на іншу при не достачі місця. За замовчуванням всі елементи вистраюються в одну строку та заповнюють весь доступний простір контейнеру. У випадку фіксованої ширини контейнеру, флекс-елементи беззсовісно вилазять за його межі.

nowrap Значення дефолтне. Флекс-елементи не переносяться, та вистраюються в одну строку
wrap Елементи переносяться, розташовуючись у кілька строк, у випадку переповнення контейнеру
wrap-reverse Флекс-елементи переносяться, але розташовуються у зворотньому напрямку та пренос відбувається знизу-вверх.
initial Встановлює значення за замовчуванням
inherit Наслідує від батьківського елементу

Синтаксис:

align-content багатострокове вирівнювання

Властивість вирівнює строки по вертикалі у флекс-контейнері, дозволяючи керувати вільним простором. Увага!!! Дана властивість працює тільки в тому випадку, якщо перенос строк дозволений, вказаний напрямок головної осі та задана висота контейнеру.

stretch Значення дефолтне. Строки розтягуються, заповнюючи весь доступний простір.
flex-start Строки флекс-елементів вирівнюються по лівому краю контейнеру, відносно верхньої сторони.
flex-end Строки флекс-елементів вирівнюються по лівому краю контейнеру, відносно нижньої сторони.
center Строки флекс-елементів вирівнюються по середині та по висоті контейнеру, відносно його лівого краю.
space-between Строки флекс-елементів вирівнюються по середині та по висоті, відносно лівого краю контейнеру. Вільний простір розподіляється між елементами. Перший ряд "піджимається" до верху контейнеру, останній до низу.
space-around Строки флекс-елементів рівномірно розділяють між собою прстір по висоті, разом з відступом від країв контейнеру
initial Встановлює значення за замовчуванням
inherit Наслідує від батьківського елементу

Властивість визначає порядок у якому будуть відображатись флекс-елементи. По дефолту заданий порядок дорівнює: 0.

Числове значення Властивість встановлюється числовим значенням, яке відповідає за порядок відображення. По замовчуванню: 0. Від'ємні значення допускаються
initial Встановлює значення за замовчуванням
inherit Наслідує від батьківського елементу

Синтаксис:

flex-basis базова ширина елементу

Властивість дозволяє задати базову ширину флекс-елементу, відносно якої буде відбуватись його розтягнення flex-grow, або ж його звуження flex-shrink.

auto Дефолтне значення. Елемент отримує базову ширину в залежності від його контенту.
Числове значення Ширина елементу задається у px rem %, та інших одиницях виміру
initial Встановлює значення за замовчуванням
inherit Наслідує від батьківського елементу

Одна властивість для задання базової ширини та трансформації елементу: flex

Властивість являє собою короткий запис трьох властивостей: flex-grow, flex-shrink, flex-basis. Дефолтне значення flex: 0 1 auto;. Задається числовим значенням або ключовими словами none, auto

Перше значення - коефіціент розширення Натуральне ціле, або дробне число, яке встановлює коефіціент збільшення флекс-елементу. Дефолтне значення: 0.
Друге значення - коефіціент звуження Натуральне ціле, або дробне число, яке встановлює коефіціент звуження флекс-елементу. Дефолтне значення: 0.
Третє значення - базова ширина Приймається у одиницях виміру. Задає за базову ширину елементу, відносно якої буде відбуватись звуження/збільшення
auto Еквівалент до flex: 1 1 auto;
none Еквівалент до flex: 0 0 auto;
