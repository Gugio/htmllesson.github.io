@media

@media запити, застосовуються для визначення типу приладу, для якого будуть застосовані стилі. В залежності від параметрів заданих у запиті, будуть застосовуватись задані для них стилі. Медіа-запити додаються в самому кінці таблиці стилів, для надання їм вищого пріорітету.

Що таке @media запити?

@media запит складаться з ключового слова, яке описує тип пристрою та виразу. З поміж усіх виразів частіше використовуються min-width та max-width. Медіа запит, це логічний вираз який повертає true або false

Логічні опеатори

За допомогою логічних операторів можливо задати комбіновані медіа-запити, у яких буде перевірятись відповідність декільком умовам

Оператор and пов'язує різні умови:

Стилі цього запиту будуть застосовуватись для всіх пристроїв ширина області перегляду яких варіюється від 768px до 980px

Правило: @media all and (max-width: 800px) { /* CSS-стилі */; }, буде рівнозначне правилу: @media (max-width: 600px) {...} (без вказання типу пристрою, застосовується по замовчуванню all)

Оператор кома:

Оператор коми працює по аналогії з оператором or

У цьому прикладі стилі спрацюють тільки для екранних або проекторних пристроїв

Оператор not:

Оператор not дозволяє спрацювати медіа-запиту у протилежному випадку. Додається на початку медіа-запиту, та застосовується для нього цілком.
Тобто запит з прикладу, буде еквівалентним запиту @media not (all and (width: 800px)) {...}

Якщо запит складений з використанням коми, то оператор not буде застосовуватись до частини яка вказується перед комою.

Типи пристроїв

Значення Опис
all всі типи пристроїв
print Використовується для документів, які переглядаються на екрані у режимі попереднього перегляду для друку
screen Використовується для екранів
speech Використовується для синтезаторів голосу

Характеристики пристроїв

До характеристик пристроїв відносяться параметри які перевіряється.

Парвметр Опис
width Перевіряє ширину області перегляду. Значення задаються у одиницях виміру px/rem/em/... наприклад (width: 768px). min-width Застосовує стилі, якщо ширина області перегляду більша за становлене значення max-width Застосовує стилі, якщо ширина області перегляду менша за становлене значення
height Перевіряє висоту області перегляду. Значення задаються у одиницях виміру px/rem/em/... наприклад (height: 320px). min-height Застосовує стилі, якщо ширина області перегляду більша за становлене значення max-height Застосовує стилі, якщо ширина області перегляду менша за становлене значення
acpect-ratio Перевіряє співвідношення сторін області перегляду. Наприклад 16:9, визначається так: (aspect-ratio: 16/9) min-aspect-ratio Перевіряє мінімальне співвідношення max-aspect-ratio Перевіряє максимальне співвідношення
orientation Перевіряє орієнтацію області перегляду. Приймає два значення: (orientation: portrait) (orientation: landscape)
resolution Перевіряє роздільну здатність пристрою. Значення також може перевіряти к-сть точок на дюйм (dpi), або на сантиметр (dpcm). (resolution: 300dpi) max-resolution Максимальна роздільна здатність min-resolution Мінімальна роздільна здатність
color Перевіряє кількість біт на кожен кольоровий компонент.Наприклад (min-color: 4) означає, що екран пристрою повинен мати 4-бітну глибину кольору max-color Максимальна Максимальна к-сть біт min-color Мінімальна к-сть біт
